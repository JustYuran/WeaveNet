WeaveNet ChangeLog v1.1.9
=========================
Дата: 2024-01-XX

Выполненные задачи:
-------------------

[+] 1. Реализовано объединение пересекающихся зон покрытия
    - Проблема: При установке нескольких вышек их зоны покрытия отображались как отдельные круги
    - Решение:
      * Добавлен новый метод drawCoverageMerged() для отрисовки объединённых зон
      * Метод проверяет каждую точку окружности на попадание в другие зоны
      * Пунктирная линия не рисуется в местах пересечения зон
      * Создаётся эффект единой объединённой зоны вместо нескольких отдельных кругов
    - Файл: /workspace/WeaveNet/src/game.js

[+] 2. Обновлён метод draw() для использования объединённой отрисовки
    - Изменения:
      * Вместо поочерёдной отрисовки каждой зоны через drawCoverage()
      * Теперь вызывается drawCoverageMerged() для всех зон сразу
      * Это позволяет корректно обрабатывать пересечения между зонами
    - Файл: /workspace/WeaveNet/src/game.js

Технические детали:
------------------
- Новый метод drawCoverageMerged() собирает все активные зоны покрытия в массив
- Для каждой зоны проверяется каждая точка окружности с шагом 2 градуса
- Если точка попадает в радиус другой зоны, она не рисуется
- Линии рисуются сегментами: beginPath() -> moveTo() -> lineTo() -> stroke()
- Шаг проверки 2° обеспечивает баланс между точностью и производительностью
- Сохранены цвета зон: синий (#4a9eff) для роутеров, оранжевый (#f59e0b) для вышек
- Пунктирная линия: setLineDash([5, 5]) с непрозрачностью 50%
