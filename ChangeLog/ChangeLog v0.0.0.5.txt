ChangeLog v0.0.0.5
Дата: 2024-04-29

Изменения:
[+] Добавлены скрипты запуска локального сервера (run_game.sh для Linux/macOS, run_game.bat для Windows)
[+] Создан README.md с инструкциями по запуску и устранению проблем безопасности браузеров
[+] Реализована автоматическая проверка наличия Python перед запуском сервера
[+] Добавлена поддержка настраиваемого порта через аргумент командной строки
[#] Исправлена ошибка "Unsafe attempt to load URL" при запуске через file:// протокол
[#] Устранена проблема с безопасностью браузеров путём использования HTTP сервера вместо прямых файловых путей

Технические детали:
- Скрипты автоматически определяют директорию игры и проверяют наличие index.html
- При отсутствии Python выводится понятное сообщение с ссылкой на скачивание
- Поддержка портов: по умолчанию 8080, можно изменить передачей числа как аргумента
- Совместимость: Python 3 (предпочтительно) и Python 2.7
