[+] 1. Исправлена синтаксическая ошибка в game.js:172
    - Проблема: Uncaught SyntaxError: Unexpected token '{' (at game.js:172:41)
    - Причина: Несоответствие типа объекта 'Tower' в drawObjectIcon() с фактическими 
              значениями 'Роутер' и 'Вышка', используемыми в остальном коде
    - Решение: Добавлена полная проверка для обоих типов объектов:
      * if (object.type === 'Роутер') - для роутеров
      * else if (object.type === 'Вышка') - для вышек
    - Файл: /workspace/WeaveNet/src/game.js (строки 169-175)
    - Проверка: node --check подтверждает отсутствие синтаксических ошибок

[+] 2. Устранены проблемы с кодировкой UTF-8
    - Проблема: Сервер отдавал файл с некорректной кодировкой русских символов
