WeaveNet ChangeLog v1.2.1
=========================
Дата: 2024-01-XX

Выполненные задачи:
-------------------

[+] 1. Добавлены всплывающие подсказки (tooltips) для элементов интерфейса
    - Проблема: Игроку было неочевидно назначение некоторых элементов интерфейса
    - Решение:
      * Создана CSS-система tooltips с анимацией появления
      * Добавлены подсказки для всех кнопок скорости (Пауза, 1x, 2x, 5x)
      * Добавлены подсказки для ресурсов (Информация, Энергия)
      * Добавлены подсказки для таймера времени игры
      * Добавлены подсказки для кнопки закрытия контекстной панели
      * Добавлены динамические подсказки для кнопок постройки (Роутер, Вышка)
      * Добавлены контекстные подсказки для кнопок управления режимами и сноса
    - Файлы: /workspace/WeaveNet/index.html, /workspace/WeaveNet/src/game.js

Технические детали:
------------------
- Реализовано через CSS псевдоэлементы ::before и ::after
- Атрибут data-tooltip хранит текст подсказки
- Класс .tooltip активирует показ подсказки при наведении
- Плавная анимация появления (opacity + transform)
- Тёмный фон подсказок с тенью для лучшей читаемости
- Динамическая генерация подсказок в game.js для кнопок с контекстно-зависимым текстом
