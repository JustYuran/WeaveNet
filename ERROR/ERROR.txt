Ошибка 1 - наложение интерфейса. 
Вылезающие окна выходят за рамки, окна накладываются друг на друга, текст выходит за рамки.
ИСПРАВЛЕНО в версии 1.2.2:
- Добавлены ограничения max-width/max-height для контекстной панели
- Улучшены стили кнопок и текста для предотвращения наложения
- Tooltip теперь поддерживают перенос длинного текста
- Все элементы интерфейса теперь корректно отображаются в пределах экрана

ИСПРАВЛЕНО в версии 1.2.3:
- Исправлено позиционирование всплывающих подсказок (tooltips) которые показывались за пределами окна браузера
- Изменён position: absolute на position: fixed для tooltip::after и tooltip::before
- Изменено позиционирование с bottom: 100% на top: 0 с transform: translate(-50%, -100%)
- Добавлено display: inline-block для .tooltip чтобы элементы корректно позиционировались
- Увеличен z-index с 1001 до 10000 для гарантированного отображения поверх всех элементов
- Добавлено pointer-events: none для tooltip чтобы подсказки не перекрывали взаимодействие
- Добавлено max-width: min(280px, calc(100vw - 20px)) для ограничения ширины относительно viewport
