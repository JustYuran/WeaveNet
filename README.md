# WeaveNet
(AI-project) Браузерная стратегия, где вы строите mesh-сети в эпоху цифровой изоляции. Размещайте узлы, оптимизируйте маршруты и балансируйте нагрузку. Обходите глушилки, чините обрывы и обеспечивайте связь. Задача — достичь нужного покрытия и пропускной способности. Процедурные миссии, управление графами и наглядная визуализация данных.
